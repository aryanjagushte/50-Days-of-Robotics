
Serial communication enables:
- ROS 2 sends motor commands to Arduino
- Arduino receives and executes commands

## Connecting Arduino to ROS 2 Node

After writing Arduino code, you connect it with the ROS 2 node.

### Connection Steps:

1. **Connect Arduino** via USB to computer running ROS 2
2. **Identify Serial Port**: `/dev/ttyUSB0` or `/dev/ttyACM0`
3. **Create ROS 2 Node**: Python or C++ node that communicates with serial
4. **Set Up Serial Library**: Use `pyserial` (Python) or `serial` (C++)
5. **Subscribe to Commands**: ROS 2 topic for motor commands
6. **Publish to Arduino**: Send commands via serial
7. **Test Communication**: Verify Arduino responds correctly

### ROS 2 Node Example (Python):

```python
import serial
import rclpy
from rclpy.node import Node
from std_msgs.msg import String

class ArduinoMotorController(Node):
    def __init__(self):
        super().__init__('arduino_motor_controller')
        
        # Initialize serial connection
        self.serial_port = serial.Serial('/dev/ttyUSB0', 9600)
        
        # Create subscriber for motor commands
        self.subscription = self.create_subscription(
            String,
            'motor_command',
            self.command_callback
        )
    
    def command_callback(self, msg):
        # Send command to Arduino via serial
        self.serial_port.write(msg.data.encode())
        
rclpy.init()
node = ArduinoMotorController()
rclpy.spin(node)
```

## Controlling Motors Through ROS 2 Commands

Once connected, you control motors through ROS 2 commands.

### Command Flow:

1. **User publishes** motor command to ROS 2 topic
2. **ROS 2 node receives** command
3. **Node sends** command via serial to Arduino
4. **Arduino parses** command
5. **Arduino controls** motor driver
6. **Motors move** accordingly

### Example Command:

```bash
ros2 topic pub /motor_command std_msgs/String "data: 'forward 100'"
```

This sends:
- Direction: forward
- Speed: 100 (PWM value)

Arduino receives and executes:
- Sets motor pins for forward direction
- Sets PWM for speed 100
