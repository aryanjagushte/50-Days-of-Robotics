## What is Gazebo Sim (Harmonic)?

Gazebo Sim (specifically **Gazebo Harmonic**) is the **new version of Gazebo**.

Key characteristics:
- It is the successor to Gazebo Classic
- It maintains similar simulation capabilities
- It has ROS 2 integration built-in
- It is actively developed and maintained
- It represents the future of robotics simulation
- It is designed for modern robotics workflows

Gazebo Sim is what the robotics community is moving toward.

## Simulation Works Similarly to Classic

The video demonstrates that **simulation in Gazebo Sim works similarly to Gazebo Classic**.

This means:
- URDF models can be loaded in Sim
- Physics simulation is similar
- Robot spawning works the same way
- Plugin concepts are comparable
- ROS 2 integration is maintained
- The workflow is familiar to Classic users

This similarity makes migration much easier than expected.

## ROS 2 Integration

The video emphasizes that **ROS 2 helps a lot in robotics engineering**.

ROS 2 benefits:
- Makes the robotics journey easier
- Provides strong integration with Gazebo Sim
- Enables control and navigation stacks
- Supports sensor simulation
- Allows visualization in Rviz2
- Provides a complete robotics framework

ROS 2 is essential for modern robotics engineering and works well with Gazebo Sim.
